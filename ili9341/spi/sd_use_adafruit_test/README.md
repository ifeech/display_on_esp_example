# SD card built into the display test

Use Adafruit lib

## Connecting the display to the esp32

| Display       | ESP32 |
|---------------|-------|
| VCC           | 3.3V  |
| GND           | GND   |
| LED           | 4     |
| TFT_MISO      | 19    |
| SD_MISO       | 19    |
| TFT_MOSI      | 23    |
| SD_MOSI       | 23    |
| TFT_CLK       | 18    |
| SD_CLK        | 18    |
| TFT_CS        | 15    |
| TFT_DC        | 16    |
| TFT_RST       | 17    |
| SD_CS         | 5     |
